Read This First

  • Voice Runtime does not know LangGraph, MCP, MoveIt, or robot policy internals.
  • Agent Control satisfies the Agent Turn backend seam and uses Robot Control.
  • Robot Control exposes structured robot tool feedback to Agent Control.
  • MoveIt planning and execution is the movement-safety boundary.

External seams and advisory context

Boundaries

The agent has explicit seams into MoveIt robot execution and Vizor user sensing. Both provide context to the agent, but only MoveIt planning/execution is the movement-safety boundary.

MoveIt MCP, Vizor User Sensing MCP, UR10 simulation

What The Agent Should Remember

  • Movement is routed through MoveIt workflows; broad ROS control stays out of Agent Orchestration by default.
  • User sensing helps resolve references like "this", "that", "there", and "near me".
  • Missing, stale, or low-confidence user sensing is not proof of user intent.
  • RViz visualizes the planning scene; it is not where agent decisions are made.
